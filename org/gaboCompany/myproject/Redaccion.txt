conceptos a explicar:
variable=referencia de datos reasignable
funciones=algoritmos, osea pasos, para hacer tareas (una única tarea, a poder ser)
recursividad=algoritmo que se ejecuta a si mismo
bucles=iteración condicional (?)
condiciones=en contexto de programación; lógica a cumplir o verificar
json=estructura de datos tipo diccionario
xml=estructura de datos para enviar paquetes de mensajes (?)
html=lenguaje de etiquetas (?) para estructurar información web
clases=objetos con propiedades y funciones 
herencia=clases que ceden, en una dirección o ambas, sus propiedad y funciones
sql=lenguaje de consulta y estructuración de datos y relaciones
queries=consulta a datos
commit=aplicar cambios a una base de datos
pull request=supongo que preguntar si se puede subir 
API=entorno(?) para recibir y enviar datos; un medio para conectar aplicaciones
